Text Basics:


In order to add a single line break in the text use the br tag. It does not require a start or a finish just insert the tag.


To add a link into your text use the a tag along with the "href=" attribute to declare the link.


To bold text within a paragraph use the b tag and insert your word/phrase between the start and end tags.


To add color to your text type style = "color:colorname within the heading or paragraph's beginning tag.


To add background colour to a paragraph, heading or body. Insert the style = into the first tag, then instead of color type background colour, it will highlight your text, heading or the whole webpage.
